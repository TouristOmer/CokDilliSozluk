# C# Multi-Language Translation & Dictionary (Console App) 🌍

This project is a simple, fast, and multi-language console translation application developed using C# and .NET Core, powered by the MyMemory Translation API. It performs asynchronous (async/await) text translation between user-defined source and target languages.

## 🚀 Features

- **Dynamic Language Selection:** Users can freely define the source and target languages at the start of the application.
- **MyMemory API Integration:** Utilizes a free translation service that does not require an API key.
- **Asynchronous Requests:** Fast, non-blocking web requests using `HttpClient` and `System.Text.Json`.
- **Continuous Execution Loop:** Allows for back-to-back translations without needing to restart the program for every new word.

## 🛠️ Technologies Used

- **Language:** C#
- **Framework:** .NET Core / .NET 6+ 
- **Libraries:** `System.Net.Http`, `System.Text.Json`
